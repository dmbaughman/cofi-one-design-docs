Capital One Investing CSS Framework

One Design

This framework implements the styles from the One Design Visual Language for use with Capital One Investing web properties. It also includes other styles and components that are useful for modern web applications.

This site is intended to provide developers with the resources they need to build web pages using this framework. This guide covers everything from One Design (as of October 1st, 2015) as well as some additional patterns that are used throughout the Capital One Investing marketing pages.

Built with Bootstrap (v3.3.5)

Bootstrap is by far the most commonly used CSS framework in the world, and with that popularity comes a wealth of shared knowledge and open source community resources. This framework applies the One Design styles to Bootstrap using its core variables pre-compilation, so the number of override styles necessary are limited.

If you haven't already done so, take some time to read through the Bootstrap Documentation to see what else you can do with this feature-rich framework.

Usage options

This framework can be used as a pre-compiled bundle or pre-compilation using select scss modules.

Using pre-compiled CSS

  • one-design-core.css only includes styles found in this style guide.
  • one-design.css includes everything from the core file, plus everything else that comes with Bootstrap.

Using with SCSS

To use with SCSS, simply import the partials you need into your project file. Refer to one-design-core.scss as an example of how to do this correctly

Here are a few tips on working with these partials:

  • Always include variables, mixins, normalize, and scaffolding files from Bootstrap
  • If importing multiple partials, maintain the same order found in _bootstrap.scss or one-design-core.scss
  • Use the custom version of _variables.scss and import it before anything else
  • Where applicable, include both the Bootstrap and custom version of each file (e.g. _buttons.scss) so the Bootstrap styles and our overrides get compiled together
