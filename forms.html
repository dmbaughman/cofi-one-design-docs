Forms

Basic form

Basic forms include input fields with a label for each field and a submit button.

<form>
    <div class="form-group">
        <label for="exampleInputEmail1">Email address</label>
        <input type="email" class="form-control" id="exampleInputEmail1" placeholder="Email">
    </div>
    <div class="form-group">
        <label for="exampleInputPassword1">Password</label>
        <input type="password" class="form-control" id="exampleInputPassword1" placeholder="Password">
    </div>
    <button type="submit" class="btn btn-primary">Submit</button>
</form>

Input types

Use input types to add semantic clarity and present mobile users with phone-friendly input options. For example, tel gives mobile users a phone-style keyboard, password masks the entered characters, and search changes the submit button from 'go' to 'search'.

<div class="form-group">
    <input type="text" class="form-control" placeholder="Text">
</div>
<div class="form-group">
    <input type="email" class="form-control" placeholder="Email">
</div>
<div class="form-group">
    <input type="tel" class="form-control" placeholder="Phone">
</div>
<div class="form-group">
    <input type="password" class="form-control" placeholder="Password">
</div>
<div class="form-group">
    <input type="search" class="form-control" placeholder="Search">
</div>
<div class="form-group">
    <input type="url" class="form-control" placeholder="URL">
</div>
