Grid

This grid provides the building blocks for a responsive application. Below is an overview of the different parts of the grid and how they work together.

Structure

The grid consists of containers, rows and columns.

  • Containers set the page margins and can contain one or more rows.
  • Rows hold one or more columns.
  • Columns are used create the layout of a page or component.

Mobile First

This grid system is mobile-first, which means that by default the layouts established for mobile will apply to all other screen sizes. From there, you can specify column widths for the larger screen sizes, if desired.

Fixed vs. fluid container

The container class applies a fixed width per breakpoint to the container. Use container-fluid to enable fluid containers.
