Backgrounds

These background classes are useful for page mastheads or other colored blocks throughout the page. The text and link colors are configured to be light or dark, depending on the background.

Pro tip: Be sure to use a .container class inside the .bg-{color-name} if you want the backround color to span the width of the page, but still have the text constrained to the container. Most of the examples below demonstrate this usage.
